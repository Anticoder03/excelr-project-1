

# 🌮 Little Taco Shop Website

## 📌 Project Overview

The **Little Taco Shop Website** is a basic multi-page website built using HTML. This project was assigned by **Radha Ma’am** from the ExcelR training program as part of the web development training. The website provides information about the taco shop, including menu, store hours, and contact details.

This project demonstrates the use of semantic HTML, navigation between multiple pages, tables, forms, images, and basic website structure.

The project includes the following pages:

* Home Page
* Store Hours Page
* Contact Page

Main homepage structure and content are defined in the main HTML file. 
Store hours information is provided on a separate page. 
Contact form and location details are available on the contact page.

## 📄 Pages Description

### 1. Home Page (index.html)

The home page contains:

* Website header
* Navigation menu
* About section
* Taco trivia section
* Menu table with prices
* Footer

It uses semantic tags like:

* `<header>`
* `<nav>`
* `<main>`
* `<article>`
* `<aside>`
* `<footer>`
* `<table>`

---

### 2. Store Hours Page (hours.html)

This page displays:

* Shop opening hours
* Navigation menu
* Image
* Footer

---

### 3. Contact Page (contact.html)

This page includes:

* Contact form (Name, Email, Message)
* Shop address
* Phone number
* Navigation menu
